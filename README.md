# go-cloudrun-boilerplate
This repository is a boilerplate for go project in STUDIO.

# Pre Condition
- Go 1.17.0 >=
- Docker 3.6.0 >=

# Quickstart
1. Initialize project 
   ```
   go mod init go-cloudrun-boilerplate
   go mod tidy
   ```
   
# Tips
## How to format all go files
```
go fmt ./...
```
